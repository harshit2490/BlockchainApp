## Blockchain App using Cryptography Algorithm
```
The sole objective of this project is to implement the cryptography algorithm used in blockchain and display them in a suitable format in the application. 
i.e. ----> blocks
     ----> previousHash
     ----> currentHash
     ----> timeStamp
```
